# 🏓 Ping-Pong

A simple implementation of an endless two-player (1v1) ping pong game.

This project focuses on core gameplay mechanics, responsiveness, and a clean minimal design.


# 🚀 Features

- Local 2-player gameplay
- Endless match (no hard stop conditions)
- Lightweight and easy to run
- Clean and straightforward game logic



# 📦 Installation

Clone the repository and install dependencies:

```bash
pip install -r requirements.txt
```
